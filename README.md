## HTML & CSS

- HTML의 태그에 대해 알고 있다.

  - [HTML 시작하기 - MDN](https://developer.mozilla.org/ko/docs/Learn/HTML/Introduction_to_HTML/Getting_started)

- 시맨틱 태그를 쓰는 이유에 대해 알고 있다.

  - [HTML text fundamentals - MDN](https://developer.mozilla.org/ko/docs/Learn/HTML/Introduction_to_HTML/HTML_text_fundamentals)

- id와 class 선택자의 차이점에 대해 설명할 수 있다.

  - [선택자의 종류 - 생활코딩](https://opentutorials.org/course/2418/13347)

- CSS box model에 대해 이해하고 활용할 수 있다.

  - [박스모델 - 생활코딩](https://opentutorials.org/course/2418/13404)

- CSS position에 대해 이해하고 활용할 수 있다.

  - [포지션 - 생활코딩](https://opentutorials.org/course/2418/13414)

- CSS 레이아웃을 원하는 위치에 배치하고 활용할 수 있다.

  - [flex - 생활코딩](https://opentutorials.org/course/2418/13526)

- CSS framework를 사용해본 적이 있다.

    - [Tailwind 공식 사이트](https://tailwindcss.com/)

- 반응형 디자인과 미디어 쿼리를 활용할 수 있다.

    - [media query - 생활코딩](https://opentutorials.org/course/2418/13517)

- SASS, SCSS와 CSS의 차이점을 알고 써본 경험이 있다.

    - [sass 공식 사이트](https://sass-lang.com/guide)

- CSS in JS의 개념을 알고 써본 적이 있다.

    - [CSS-in-JS에 관해 알아야 할 모든 것](https://d0gf00t.tistory.com/22)
